object 를 string 으로 바꾸는 방법은 `JSON.stringify()`를 사용한다.

``` js
const obj = [{name: 'first'}, {name: 'second'}, {name: 'third'}]

JSON.stringify(obj) // "[{"name":"first"},{"name":"second"},{"name":"third"}]
```
