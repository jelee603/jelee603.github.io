#### CSS 그리드를 사용할 때, 중첩된 그리드는 어떻게 나눌까?
아래와 같이 하면 만들어진 main 안에 content1, content2 가 나눠진다. 
 
``` js
.main {
    display: grid;
    grid-rows: 1/10;    
    template-rows: repeat(10, 1fr);
    grid-gab: 5px;    
}

.content1 {
    grid-rows: 1/6;
}

.content2 {
    grid-rows: 6/10;
}
```
