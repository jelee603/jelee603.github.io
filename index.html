Mac을 편하게 사용하는 방법 7가지

Mac 단축키

option + cmd + esc : 응용 프로그램 강제 종료
option + cmd + i : 크롬 개발자 도구
cmd + space : 스포트라이트 검색 - (활성 상태 보기)
cmd + w : 크롬 텝 닫기
cmd + y : 크롬 브라우저 방문기록 보기
cmd + shift + 4 : 영역지정 캡쳐
