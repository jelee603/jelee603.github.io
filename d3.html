D3 필요한 메서드 정리해보자

2019-10-01

D3 차트에서 가장 많이 쓰이는 속성

transform 속성은 Axis를 그릴 때, call 함수로 데이터를 바인딩하면서 간격을 지정해준다. translate(x, top) : x 축과 top 위치가 시작점

D3 라인 그리는 방법

var svg = d3
  .select('#segment')
  .append('svg')
  .attr('width', 800)
  .attr('height', 200);

svg
  .append('line')
  .attr('x1', 10)
  .attr('y1', 10)
  .attr('x2', 700)
  .attr('y2', 100)
  .attr('stroke', 'red');
