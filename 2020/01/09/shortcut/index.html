VSCode 단축키 (Window 단축키)

cmd + d (ctrl + d) : 같은 파일내 단어 찾기
option + cmd + i (F12) : 크롬 디버그
ctrl + space : 파일에서 프로퍼티 자동완성
cmd + shift + p : 페이지 셋업
cmd + p : 파일 찾기
cmd + j : 터미널 창 열기
